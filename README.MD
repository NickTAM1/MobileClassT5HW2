# AR Treasure Collector

A simple AR mini game built with Unity AR Foundation.

## How to Play
1. Launch the app on an ARCore compatible Android device (or ARKit iOS device).
2. Slowly move your phone to scan the room. Detected surfaces appear as
   translucent blue outlines.
3. Tap any detected surface to drop a gem there.
4. Tap a gem to collect it and increase your score.
5. If tracking is lost, the on screen message will tell you to move
   slowly to a brighter or more detailed area.

## Technical Setup
- Unity AR Foundation 6.3.5
- ARCore XR PlugIn (Android)
- Built with: AR Session, XR Origin (AR Camera, AR Plane Manager,
  AR Raycast Manager), and `ARPlacementManager.cs` for game logic.
